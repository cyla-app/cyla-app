# Cyla

# Setup

## backend

1. `cd backend/`
1. `go get -d`

## frontend

1. `cd frontend`
1. `npm instal`
1. `npx react-native start`

## Docker

1. Create a `.env` file in the project root. Fill it according to `.env.example`.
1. Start all containers: `sudo docker-compose up`
